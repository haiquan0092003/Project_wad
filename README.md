## 🚀 Installation and Setup

### Prerequisites
- Node.js >= 14.0.0
- MongoDB >= 4.0
- npm >= 6.0.0

### Quick Start
```bash
# Clone repository
git clone [repository-url]

# Install dependencies
npm install

# Run the project
npm start
```
