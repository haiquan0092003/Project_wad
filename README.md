## 🚀 Installation and Setup

### Prerequisites
- Node.js >= 14.0.0
- MongoDB >= 4.0
- npm >= 6.0.0
- Docker (optional)

### Quick Start
```bash
# Clone repository
git clone [repository-url]

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Run the project
npm start
```

### Docker Setup
```bash
# Build and run with Docker
docker-compose up --build
```
